# Velocibet-OS

Velocibet-OS(가칭)은, 10년을 잡아 OS를 개발하는 장기개발 프로젝트입니다.

**실제 상용화가 목적이 아닌, 개인이 개발하는 능력의 한계를 시험해보기 위한 목적으로 제작합니다.**

## 개발 목표
> 무(無)에서 유(有)를 창조합니다.

### 개발 인프라

- VGA text buffer
- Serial output
- Panic 기능
- Memory dumper
- 어셈블리어 및 링커
- 디스어셈블리어
- 프로그래밍 언어
- 자체 컴파일러

### OS 개발

- 부트로더
- CPU 제어
- 메모리 관리
- 프로세스 스케쥴링
- 저장장치 드라이버
- 입출력 드라이버
- 파일 시스템
- 쉘 명령어
- framebuffer
- pixel 렌더링
- 관련 네트워크 기능

### 셀프호스팅

Rust로 컴파일러 작성 후 후에 제작한 언어로 재작성을 목표

## 기타

AI를 활용하여 개념을 학습하는 것은 가능하지만, AI가 짜준 코드를 복사 붙여넣기 하는 바이브 코딩은 일절 하지 않습니다.
